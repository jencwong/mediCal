# mediCal(Back-end)
  Backend server for the MediCal App

## Technologies
* Node.js
* Express
* MongoDB
* Bcrypt
## (User) Features
* Sign-in/Sign-up w/ encypted password 
* Add Appointment
* Update Appointment
* View all upcoming user appointments
## (Admin) Features
* Sign-in/Sign-up w/ encypted password 
* View all upcoming appointments
* Delete any appointment
